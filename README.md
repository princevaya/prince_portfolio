# Premium Futuristic Cloud Portfolio

Premium, cinematic portfolio built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, GSAP, React Three Fiber, and Lenis smooth scrolling.

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000
